# Flutter Authentication App

A modern Flutter app for user authentication with a Flask backend and MongoDB.

## Features
- User registration and login
- JWT-based session management
- Modern Material Design UI
- Provider for state management
- Secure token storage
- Light/Dark theme support

## Getting Started
1. Install Flutter (latest stable): https://docs.flutter.dev/get-started/install
2. In this directory, run:
   ```sh
   flutter pub get
   flutter run
   ```
3. Ensure the backend is running at `http://localhost:5000`.

## Packages Used
- provider
- http
- flutter_secure_storage
- shared_preferences
- email_validator
- fluttertoast

---

See `/backend` for the Flask API.
